🌿 Harvestify: The Beginner’s Gap Solution - PROJECT NUMBER 3

An interactive, responsive Hydroponic Growing Platform designed and built for those looking to turn every home in the world into a micro-farm. This tool demonstrates the ease of transitioning from traditional, intimidating soil-based gardening to Harvestify’s streamlined, automated hydroponic system.

✨ Features
Harvestify Crop Finder: An intuitive, responsive interface that replaces endless research with a four-question diagnostic. It processes user space, light, experience level and liking to generate a tailored growing plan that is guaranteed to thrive.
The Harvestify Crop Guide: A built-in, mini chat-box interface that provides real-time answers to the user's most pressing questions regarding crops, lighting, and hydroponic maintenance.
Zero-Friction Design: A fast, lightweight, and fully responsive web application. Optimized with clean, efficient code so that anyone, anywhere in the world, can access the tool instantly on any device—no downloads or complex setups required.
Visionary Roadmap: Built as a scalable MVP, designed to evolve into a global, community-driven platform featuring user-uploaded progress photos and a crowdsourced database of growing tips.
